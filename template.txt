
     When adding content from a search API crawl, use the appropriate template based on content type.

     ---
     Quick Reference: Content Types

     | Type      | File                | Best For                           |
     |-----------|---------------------|------------------------------------|
     | Package   | data/packages.json  | Libraries, tools, frameworks       |
     | Dataset   | data/datasets.json  | Data collections, benchmarks       |
     | Resource  | data/resources.json | Blogs, tutorials, courses          |
     | Book      | data/books.json     | Published books                    |
     | Talk      | data/talks.json     | Videos, podcasts, interviews       |
     | Paper     | data/papers.json    | Academic papers (nested structure) |
     | Career    | data/career.json    | Career guides, industry insights   |
     | Community | data/community.json | Conferences, meetups               |

     ---
     Template 1: PACKAGE (Most Common)

     {
       "name": "",
       "description": "",
       "category": "",
       "url": "",
       "tags": [],
       "language": "",
       "docs_url": null,
       "github_url": null,
       "install": "",
       "best_for": ""
     }

     Required: name, description, category, url, tags, language
     Categories: Adaptive Experimentation & Bandits, Causal Inference & ML, Statistical Computing, etc.
     Languages: Python, R, Julia, JavaScript, etc.

     Template 6: PAPER (Academic)

     Papers are nested within topics/subtopics. Add to existing subtopic:

     {
       "title": "",
       "authors": "",
       "year": 2024,
       "url": "",
       "tags": [],
       "citations": 0,
       "tag": "",
       "description": ""
     }

     Required: title, authors, year, url, tags, citations
     Tag values: Classic, SOTA, Industry, Survey, Methodological

     ---
     LLM-Enriched Fields (Auto-Generated)

     These fields are added automatically by running scripts/enrich_metadata.py:

     {
       "difficulty": "beginner|intermediate|advanced",
       "prerequisites": ["prerequisite-1", "prerequisite-2"],
       "topic_tags": ["specific-topic-1", "specific-topic-2"],
       "summary": "AI-generated 2-3 sentence summary",
       "use_cases": ["use-case-1", "use-case-2"],
       "audience": ["Junior-DS", "Mid-DS", "Senior-DS", "Early-PhD"],
       "synthetic_questions": ["How do I...", "What is..."]
     }

     Do NOT fill these manually - they are generated by the enrichment script.

     ---
     Adding Content: Step-by-Step

     1. Identify content type from the table above
     2. Copy appropriate template
     3. Fill required fields (marked in each template)
     4. Use existing categories - don't create new ones unless necessary
     5. Add to appropriate JSON file in data/ directory
     6. Run enrichment (optional): python3 scripts/enrich_metadata.py
     7. Regenerate search index: python3 scripts/generate_embeddings.py
     8. Build site: hugo --gc --minify

     ---
     Validation Checklist

     Before adding:
     - URL is valid and accessible
     - Description is clear and informative (1-2 sentences)
     - Category matches existing categories
     - Tags are relevant (3-5 tags recommended)
     - No duplicate entry exists
     - JSON syntax is valid
